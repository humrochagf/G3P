**Resumo**

Este trabalho busca a criação de um produto personalizado e simplificado para uma loja de festas infantis gerenciar seus produtos, parceiros e pedidos, fazendo para isto as modelagens conceitual, lógica e física de um banco de dados o ligando a um web framework para criar a interface com o usuário tendo em mente a facilidade de operação, a integração dos parceiros com a loja, a organização dos produtos e o controle dos pedidos além de atender a todas particularidades encontradas durante a modelagem do problema.

**Palavras-chaves:** gerenciar, produtos, parceiros, pedidos, banco de dados, web framework.
